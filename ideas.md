# 学情反馈MVP - 设计方案

## 项目定位
这是一个**内部工具**，用于快速验证：输入算术题 → 调用Claude API → 保存结果到Google Drive

## 设计选择：极简工具风格

由于这是一个最小验证版本（MVP），采用**极简工具风格**：
- 清晰的功能导向
- 无干扰的界面
- 快速完成任务

### 设计要素
- **配色**：浅灰背景 + 深色文字 + 蓝色强调
- **字体**：系统默认字体，保持简洁
- **布局**：居中单列，表单式交互
- **交互**：即时反馈，状态清晰

### 页面结构
1. 标题区：简单说明功能
2. 输入区：算术题输入框
3. 操作区：提交按钮
4. 结果区：显示计算结果和保存状态

---

**选定方案**：极简工具风格，专注功能验证
