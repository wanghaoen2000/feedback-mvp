# 项目概述

## 基本信息

| 项目 | 内容 |
|------|------|
| 项目名称 | 托福阅读学情反馈系统（feedback-mvp） |
| 当前版本 | V33 |
| 项目状态 | 可用，Google Drive授权待修复 |
| 部署平台 | Manus |

## 项目定位

教学辅助自动化工具，帮助新东方托福阅读教师和助教快速生成课后文档。

## 核心功能

输入课堂信息后，自动生成5个文档：
1. 学情反馈（.md）
2. 复习文档（.docx）
3. 测试本（.docx）
4. 课后信息提取（.md）
5. 气泡图（.png）

## 运维功能

- 系统自检：一键检测8项基础条件
- 日志记录：每次生成自动记录详细日志
- 日志导出：上传到Google Drive
- 中文错误提示：API错误自动转换为中文

## 技术栈

- 前端：React 19 + TypeScript + Tailwind CSS
- 后端：Node.js + tRPC + Express + Drizzle ORM
- 数据库：MySQL
- AI服务：DMXapi
- 文件存储：Google Drive（rclone）
